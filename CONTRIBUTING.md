Ways to Contribute

There are many ways to contribute to Open Mission.

You can help by contributing to:

Software

* Report bugs
* Develop new features
* Review pull requests
* Improve accessibility
* Design user interfaces
* Test applications

Documentation

* Improve documentation
* Write tutorials
* Translate content
* Create guides

Knowledge

* Share best practices
* Propose open standards
* Write technical or ministry playbooks
* Improve project architecture
* Participate in RFCs and discussions

Community

* Answer questions
* Mentor contributors
* Review ideas
* Share feedback

Every contribution helps strengthen the Open Mission ecosystem.
